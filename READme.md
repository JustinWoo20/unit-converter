# Unit Converter 🧮

A simple GUI-based unit converter built with Python and Tkinter.

## 🚀 Features
- Converts:
  - Miles ↔ Kilometers
  - Celsius ↔ Fahrenheit
  - Centimeters ↔ Feet and Inches
- Clear and intuitive layout
- Error handling for non-numeric input
